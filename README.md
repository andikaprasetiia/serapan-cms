## Overview

Serapan CMS is a premium content management system designed for professional newsrooms, editorial teams, and media-style publishing workflows. The platform combines a Laravel backend with a modern Vue and Inertia administrative experience, giving editors a fast dashboard while keeping the public-facing site structured, responsive, and SEO-ready.

This public repository is a **showcase and documentation repository only**. The production source code, installation files, deployment scripts, database schema, and private business logic are intentionally not published.

## Highlights

- Editorial dashboard for managing posts, pages, categories, tags, and media assets.
- Rich text authoring with structured content controls.
- SEO fields for article metadata, preview snippets, and social sharing.
- Media library with upload, preview, and asset organization workflows.
- Advertisement management for multiple site placements.
- Menu builder for nested navigation structures.
- Import and export flows for controlled content portability.
- System health and backup utilities for administrative maintenance.
- Indonesian and English interface support.
- Responsive public presentation for media and publication-style websites.

## Source Code Policy

This repository does **not** include the executable application source. The following files and directories are deliberately excluded:

- Application source code
- Dependency manifests such as `composer.json` and `package.json`
- Environment files such as `.env` and `.env.example`
- Database migrations, seeders, and SQL dumps
- Docker and deployment configuration
- Build scripts and initialization scripts
- Authentication, backup, and updater implementation details
- Private assets, screenshots, credentials, and client data

Because those files are not included, this repository cannot be installed or run as an application.
