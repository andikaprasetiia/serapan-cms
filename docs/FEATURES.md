# Feature Overview

Serapan CMS is designed for editorial publishing workflows where administrators
need structured content tools, media controls, and a fast management interface.

## Content Management

- Posts and pages
- Categories and tags
- Rich text editor
- SEO metadata
- Author management
- Template-aware content presentation

## Media Management

- Asset upload workflow
- Grid and list media views
- Detail preview panel
- Centralized media organization

## Navigation

- Nested menu builder
- Custom links
- Internal content mapping
- Public navigation rendering

## Publishing Operations

- Import and export workflows
- Backup and restore utilities
- System health status checks
- Administrative dashboard metrics

## Public Site

- Responsive layout
- Advertisement placements
- Hero and feed presentation
- Category-driven browsing
- Search-oriented metadata structure
