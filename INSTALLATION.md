# Installation

Installation is not available from this public repository.

Serapan CMS is proprietary software, and the executable source code is maintained
privately. This repository does not include the files required to install, build,
or run the application.

The following materials are intentionally not distributed:

- Dependency manifests
- Application source code
- Database schema and seed data
- Environment templates
- Deployment scripts
- Docker configuration
- Build pipeline configuration

For authorized licensing or deployment access, contact the repository owner.
