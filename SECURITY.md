# Security Policy

## Supported Code

The executable Serapan CMS source code is maintained privately. This public
repository does not contain installable application code.

## Reporting Security Issues

If you believe you have found a security issue related to Serapan CMS, contact
the repository owner privately. Do not publish exploit details, credentials,
database content, or private implementation information in public issues.

## Public Repository Scope

This repository is documentation-only and intentionally excludes:

- Credentials and environment files
- Database dumps
- Application source code
- Build and deployment scripts
- Private media assets
- Dependency manifests
